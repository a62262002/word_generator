# 幹話產生器

運用 node.js 和 express 進行開發，呈現針對工程師、設計師及創業家的幹話。

## 安裝環境

- Windows 8
- Node.js v10.16.3
- Express ^4.17.1

## 安裝

1. 下載 generate_word 資料夾，並輸入以下指令下載

```
$ git clone https://github.com/a62262002/word_generator.git
```

2. 從終端機安裝 npm 套件，輸入以下指令

```
$ npm install
```

3. 執行專案

```
$ npm run dev
```

4. 輸入網址並呈現幹話產生器

```
http://localhost:3000
```

## 功能說明

- 顯示幹話清單：http://localhost:3000
